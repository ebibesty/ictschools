TOPIC ONE: DATABASE

A database is an organized collection of related data kept in a format that makes it easy to retrieve, access, modify, process and store. An example of simple database is an address book or a list containing studentsâ€™ results.


A database package, also known as Database Management System (DBMS) is the term used to describe any computer software that is designed for creating databases and managing the information stored in them i.e. it manipulates, retrieves, and manages data in a database as well as interacts with the users and other applications. DBMS allows users create, query, update and administer databases. It also helps to maintain the integrity and security of stored data.
